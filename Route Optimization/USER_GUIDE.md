# User Guide: Food Truck Route Optimizer

## Quick Start

### 1. Set Your Start Location
- Click **"ðŸ“ Use Current Location"** to use your GPS location
- Or click on the map to set a custom start location

### 2. Add Delivery Addresses

#### Method 1: Search Bar (Recommended - Fastest!)
1. **Look for the search bar at the top of the page**
2. **Type the delivery address** (e.g., "123 Main St")
3. **Autocomplete suggestions appear** as you type (after 3+ characters)
4. **Select a suggestion**:
   - **Click** on a suggestion
   - **Use Arrow Up/Down** keys to navigate, then **Enter** to select
   - Or **continue typing** and press Enter
5. The address is automatically converted and added as a stop
6. **Repeat for all delivery addresses**

**Features:**
- Real-time autocomplete suggestions (Google Places API)
- Location-biased results (prioritizes addresses near your start location)
- Keyboard navigation (Arrow keys, Enter, Escape)
- Recent addresses shown below for quick re-entry
- Auto-completes with formatted address
- No need to enter coordinates manually

#### Method 2: Click on Map
1. Click anywhere on the map
2. A stop is added at that location
3. You can drag it to adjust position

#### Method 3: Manual Entry (Advanced)
1. Click **"+ Add Stop (Manual)"** in the sidebar
2. Enter address and click "ðŸ“ Find" to geocode
3. Or enter coordinates manually
4. Set priority, urgency, time windows if needed

### 3. Optimize Your Route
1. Once you have at least one stop added
2. Click **"ðŸš€ Optimize Route"**
3. The system will:
   - Calculate the fastest route
   - Consider traffic conditions
   - Find parking locations
   - Show vehicle and walking routes

### 4. View Your Route
- **Black lines**: Vehicle routes (truck to parking)
- **Green dashed lines**: Walking routes (parking to delivery)
- **Orange "P" markers**: Parking locations
- **Numbered markers**: Stop order

## Troubleshooting

### Address Not Found
- Check spelling
- Include city and state
- Try a more specific address

### Route Not Optimizing
- Make sure you have at least one stop
- Check that start location is set
- Wait for geocoding to complete

### Map Not Loading
- Check internet connection
- Refresh the page
- Clear browser cache
