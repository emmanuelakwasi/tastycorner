## Voice Commands

### Using Voice Input

1. **Click the microphone button** (üé§) in the chat input area
2. **Speak your question** clearly
3. Bot will **transcribe** what you said
4. Bot will **respond** with both text and voice

### Tips for Voice Input
- Speak clearly and at a normal pace
- Works best in quiet environments
- Available in Chrome and Edge browsers
- Microphone button turns red when listening

## Troubleshooting

### Chatbot Not Responding?

1. **Check if route is optimized**: Bot needs an optimized route to answer most questions
2. **Try typing instead of voice**: Text input always works
3. **Check browser console**: Look for any error messages
4. **Refresh the page**: Sometimes helps reset the chatbot

### Voice Input Not Working?

1. **Browser support**: Voice input works in Chrome and Edge
2. **Microphone permission**: Allow microphone access when prompted
3. **Check microphone**: Make sure your microphone is working
4. **Use text input**: Text input always works as a fallback

### Voice Output Not Working?

1. **Browser support**: Voice output works in all modern browsers
2. **Volume**: Check your device volume
3. **Browser settings**: Check browser audio settings
4. **Try pause/play**: Click the pause button to stop, then ask again
