# Feature Images

Add the following image files to this directory for the "Why You'll Love TastyCorner" section:

1. **fresh-ingredients.png** - Image representing fresh, high-quality ingredients (e.g., fresh vegetables, herbs, or ingredients being prepared)

2. **fast-delivery.png** - Image representing fast delivery service (e.g., delivery truck, delivery person, or clock/time symbol)

3. **secure-payment.png** - Image representing secure payments (e.g., credit card, lock icon, or payment security symbol)

4. **allergy-friendly.png** - Image representing allergy-friendly options (e.g., health symbol, checkmark, or allergy-safe badge)

5. **rated.png** - Image representing high ratings (e.g., star rating, 5 stars, or rating badge)

6. **loyalty-rewards.png** - Image representing loyalty rewards (e.g., gift box, reward badge, or points symbol)

## Image Specifications:
- **Format**: PNG (with transparent background recommended)
- **Size**: Recommended 240x240px or larger (will be scaled to 120x120px)
- **Style**: Modern, clean, and professional icons or illustrations
- **Colors**: Should work well with the site's color scheme (blues and purples)

## Fallback:
If an image is missing, the section will automatically show an emoji placeholder until you add the actual image file.
