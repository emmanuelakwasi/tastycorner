# Hero Background Image

## Instructions

1. **Add your hero background image** to this folder (`static/images/`)
2. **Name it exactly**: `hero-background.jpg`
3. **Recommended specifications**:
   - Format: JPG, PNG, or WebP
   - Size: 1920x1080 pixels or larger (for high-resolution displays)
   - File size: Under 500KB (optimize for web)
   - Aspect ratio: 16:9 or wider

## Image Suggestions

Good images for restaurant hero backgrounds:
- Beautiful food photography
- Restaurant interior/exterior
- Table setting with food
- Chef preparing dishes
- Delicious plated meals

## If Image Doesn't Load

If the image file doesn't exist, the hero section will use a blue gradient fallback color (#1e40af) so the page still looks great!

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Consider using WebP format for better compression
- Make sure the image is high quality but optimized for web
